# Folder Structure

This project includes wiki pages, source-processing folders, scripts, outputs, and reusable skills. Skills are reusable instructions that help AI agents perform specific tasks consistently and safely.

## Skills

- `skills/rest-api-workflow.md` — Instructions for safely and repeatably using an external RESTful API.
- `skills/librarycloud-search.md` — Instructions for searching Harvard LibraryCloud or a similar library catalog API.
